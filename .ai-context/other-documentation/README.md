# Other Documentation

- Use this folder only for AI-facing notes that do not clearly belong in `main-documentation`, `project-context`, or `lessons-learned`.
- Files here do not need polished formatting and can be raw archives, exports, transcripts, or other low-priority support material.
- Agents should not read this folder during normal bootstrap; only open files here when the user explicitly asks for them or a task directly depends on a specific file.
- Keep files short, factual, and reusable when practical, but raw captures are acceptable here.
- If a note becomes core project knowledge, move or merge it into `main-documentation`.
